Description

Running a movie theater is no easy business. To help our friends, let's add statistics to your app. The stats will show the current income, total income, the number of available seats, and the number of occupied seats.

Objectives

Add a simple layout to display the information about our cinema room:

  1. TextView with cinema_room_total_income id and the following text: Total cinema income: X.XX$;
  2. TextView with cinema_room_current_income id and the following text: Current cinema income: X.XX$;
  3. TextView with cinema_room_available_seats id and the following text: Available seats: XX;
  4. TextView with cinema_room_occupied_seats id and the following text: Occupied seats: X.
